# Drone X-press
* 드론을 몰고 마을에 빠르게 배달을 목표로 하는 캐쥬얼 비행 아케이드 게임
* 3개의 스테이지에 따른 난이도 설정과 디버프/장애물/기믹을 통한 다양한 이동 루트.

## 주요 기능

* 드론 비행 조작
* 장애물 충돌
* 디버프 장애물
* 가속 게이트
* 스테이지별 난이도 증가

## 플레이 설명

* 드론을 조종해 목적지까지 이동합니다.
* 장애물과 디버프를 피하고 가속게이트를 활용해 최적의 경로를 찾아 빠르게 배달을 완료합니다.
